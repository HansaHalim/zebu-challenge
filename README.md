# zebu-challenge
Make a Pizza Online App

#### Description

Screens:
- Select pizza size
- Select pizza crust
- Select Toppings
- Review Custom Pizza

There are 3 sizes:
- Small ($8)
- Medium ($10)
- Large ($12)

2 Crust Type:
- Thin (+ $2)
- Thick (+ $4)

to be continued...

## Build/Run

#### Requirements

- Node.js
- NPM

```javascript

/* First, Install the needed packages */
npm install

/* Then start both Node and React */
npm start

```
